## Reliability Rule

- `Units, sections, and position titles` can be used as current structure input.
- `Personnel names` in the charts should be treated as outdated until replaced with updated assignments.
- For the website, the safest version-one approach is `position-only structure first`.
- The structure should now be treated as `role-based`, with current names handled separately in the assignment layer.
