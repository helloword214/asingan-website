# Images Folder

You can place generated mock images here now.

Recommended folders:

- `mock-station`
- `mock-personnel`
- `mock-personnel/fire-marshals`
- `mock-personnel/personnels`
- `derived/mock-personnel/fire-marshals`
- `derived/mock-personnel/personnels`
- `mock-assets`
- `mock-gallery`
- `branding`

Notes:

- Keep uploaded originals in `mock-personnel/personnels`.
- Keep uploaded fire marshal originals in `mock-personnel/fire-marshals`.
- Generated resized display copies may be stored in `derived/mock-personnel/personnels` so the originals remain untouched.
- Generated resized fire marshal copies may be stored in `derived/mock-personnel/fire-marshals` so the originals remain untouched.

See [Asset Guide](../../docs/ASSET_GUIDE.md) for details.
